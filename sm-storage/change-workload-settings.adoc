= Change workload settings in SANtricity System Manager
:experimental:
:icons: font
:imagesdir: ../media/

[.lead]
You can change the name for a workload and view its associated application type. Change the name of a workload when the current name is no longer meaningful or applicable.

.Steps

. From the left panel, select menu:Storage[Volumes].
. Select the *Applications and workloads* tab.
+
The Applications and workloads view appears.

. Select the workload that you want to change, and then click the menu:ellipsis icon[Edit Settings].
+
The Applications and workloads Settings dialog box appears.

. *Optional:* Change the user-supplied name of the workload.
. Click *Save*.
