= Set the default units for capacity values in SANtricity System Manager
:experimental:
:icons: font
:imagesdir: ../media/

[.lead]
SANtricity System Manager can display capacity values in either gibibytes (GiB) or tebibytes (TiB).

Preferences are stored in the browser's local storage so all users can have their own settings.

.Steps

. Click the *User Settings* icon in the upper-right corner of the window. 
. In the Display capacity values in drop-down list, select either either *Gibibytes* or *Tebibytes*.
+
See the following table for abbreviations and values.
+
[cols="1a,1a" options="header"]
|===
| Abbreviation| Value
a|
GiB
a|
1,024^3^ bytes
a|
TiB
a|
1,024^4^ bytes
|===
